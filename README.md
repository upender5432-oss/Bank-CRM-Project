# 📊 Bank CRM Dashboard (Power BI)

## 📌 Overview

The **Bank CRM Dashboard** is a data visualization project built using Power BI to analyze customer data, transaction behavior, and banking performance metrics.
It helps stakeholders make data-driven decisions by providing clear insights into customer activity and trends.

---

## 🚀 Key Features

* 📊 Interactive dashboards for customer analysis
* 👥 Customer segmentation based on activity and transactions
* 💳 Transaction trend analysis
* 📈 KPI tracking (Total Customers, Total Transactions, Revenue, etc.)
* 🔍 Dynamic filtering using slicers
* 📉 Time-based analysis (monthly/yearly trends)

---

## 🛠️ Tools & Technologies

* **Power BI** (Dashboard & Visualization)
* **SQL** (Data extraction & transformation)
* **Excel / CSV** (Data source)

---

## 📊 Dashboard Insights

* Identified high-value customers contributing to maximum revenue
* Analyzed transaction frequency and customer engagement
* Detected trends in customer activity over time
* Highlighted inactive or low-performing customer segments
